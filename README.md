AI Crop Disease Detector
🌿 Project Overview
This project is an AI-powered Crop Disease Detector developed using Python. It aims to assist farmers and agricultural researchers by identifying various crop diseases from images using deep learning models. By automating the detection process, the tool helps in early intervention and crop loss prevention.

👥 Group Project Details

Python-Based: Built using high-performance Python libraries for machine learning.

Scalable Architecture: Designed to eventually support a wide variety of crops and regional diseases.
